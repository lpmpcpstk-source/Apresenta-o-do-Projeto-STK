O Problema

A regra de três falha
sistematicamente

O enrolamento radial é uma função quadrática, não linear. Cada volta percorre uma circunferência maior que a anterior — calcular por proporção simples acumula erro em todo pedido.

📉
Erro Sistemático
Proporção simples ignora o crescimento radial. O erro aumenta conforme a diferença entre diâmetro vazio e cheio da espula.
Regra de três = resultado errado sempre
🧵
Desperdício de Fio
Pulsos a mais desperdiçam fio caro. Pulsos a menos exigem paradas, inspeções e retrabalho em toda a linha de produção.
Cada parada = atraso + custo extra
🗒️
Processo Manual
Cálculos em papel ou planilhas desatualizadas geram inconsistências entre turnos, operadores e máquinas. Sem rastreabilidade.
Sem histórico, sem controle
A Solução

Matemática exata,
resultado certo

01
Modelo Quadrático Correto
Usa o modelo de crescimento radial linear (Espiral de Arquimedes) que representa com exatidão como o fio se acumula na espula.
sem erro no modelo
02
Inversão via Bhaskara
Para calcular pulsos a partir de metros, resolve a equação quadrática inversa com precisão de 6 casas decimais. Arredonda sempre para cima — nunca falta fio.
⌈x⌉ — arredondamento exato
03
Divisão em Lotes Automática
Calcula Lote 1 (espulas cheias) e Lote 2 (parciais) com os pulsos exatos para cada programação, para qualquer número de fusos.
qualquer configuração de fusos
04
Fichas Técnicas por Espula
Cada modelo tem sua ficha com parâmetros calibrados. Cadastrada uma vez, o operador só informa a metragem e os fusos.
reutilizável em todos os pedidos
Modelo Matemático — Espiral de Arquimedes
// Grandezas base
C₀ = π × D₀ / 100
C_max = π × D_max / 100
K = (M_max / C_med) / P_max

// Metragem após x pulsos
m(x) = C₀·K·x + (ΔC·K / 2·P)·x²
// Inverter: metros → pulsos
a = (ΔC × K) / (2 × P_max)
b = C₀ × K
c = −m_desejada
x = (−b + √(b²−4ac)) / (2a)
P = ⌈x⌉ // sempre para cima
✅ Validado: m(60) = 720,000 mErro do modelo = 0 m — margem de arredondamento exibida no relatório.
Fluxo de Uso

4 passos até o setup

1
Selecionar Ficha
Escolha o modelo de espula com seus parâmetros físicos já calibrados no sistema.
2
Informar Pedido
Digite a metragem total do pedido e o número de fusos da trançadeira.
3
Calcular Setup
O sistema calcula os pulsos exatos para Lote 1 (cheias) e Lote 2 (parciais) instantaneamente.
4
Programar Máquina
O operador programa os contadores Fotek com os valores exibidos no relatório.
